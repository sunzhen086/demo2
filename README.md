
### 文件目录

+ src
  - assets —— 图片等资源文件
  - components —— 通用组件
  - pages —— 页面文件夹
  - styles —— 全局通用样式文件夹
  - utils —— 工具库文件夹
  - docs —— 构建后的输出文件夹


### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
